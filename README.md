# USER-AUTH API

## Config

Example of `.env` config file in root directory:

```angular2html
PORT=5000
JWT_SECRET=secret
JWT_EXPIRATION=30s
BCRYPT_SALT=10
```

## Test data

```angular2html
login: user###,
password: 1234!@#,
```

## Installation

- `npm i`
- `npm run build`
- `npm start` or `npm run dev` to run in development mode
